# TerryBot
A multipurpose Discord bot

## Setup and Execution
1. Clone the repository:
   ```bash
    git clone https://github.com/WiiExpertise/TerryBot.git
    ```
2. Navigate to the project directory:
 
    ```bash
    cd TerryBot
    ```
3. Install dependencies:

    ```bash
    npm install
    ```
4. Rename ``config-sample.json`` to ``config.json`` and set values such as bot token and client ID

5. Run the bot:

    ```bash
    node index.js
    ```
